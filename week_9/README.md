# IT Helpdesk Ticket Registration System

## 1. Purpose of the Application
The IT Helpdesk Ticket Registration System is a modular Python application designed for City University Malaysia. It enables students to submit technical issues (e.g., login errors, Wi-Fi issues, printer problems) and automatically assigns a technician based on the issue's priority level.

## 2. Tech Stack
* **Programming Language:** Python 3
* **Version Control:** Git & GitHub
